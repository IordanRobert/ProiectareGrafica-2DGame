ProiectareGrafica-2DGame

This game was made in OpenGL with sprites downloaded from CraftPix.net

In menu use the mouse to click on the buttons.

For controlling the character in game we have the following keys:

W - jump

A - move left

D - move right

S - fall faster

J - attack 1

K - attack 2

L - special attack

ESC - exit the current menu/submenu, closes the game if pressed in the main men
